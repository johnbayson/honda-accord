HDS Clear Command
The inside ECM stores various specific data to correct the system even when there is no electrical power such as the battery negative terminal or No. 8 FI ECU (ECM) (15 A) fuse are disconnected. But stored data based on failure parts should be cleared by using the ‘‘CLEAR COMMAND'' of the HDS, if parts are replaced.
The HDS has three kings of clear command to meet the purpose. They are DTC clear, ECM clear. DTC clear command erases all stored DTC codes and Freeze Data. This must be done with the HDS after reproducing DTC during troubleshooting in this service manual. ECM clear command erases all stored DTC codes, Freeze Data and all specific data to correct the system.
