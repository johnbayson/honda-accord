NOTE:
  • When the following works, the steering angel sensor neutral position memorizing needs to be done. The memorizing is done by using HDS according to the screen display of the HDS.
  • Check the following before the memorizing operation. If there is malfunction, the memorizing operation may not be done and the adjustment or replacement may be required.
The work which needs steering angle sensor neutral position memorizing in.
  • Steering wheel removal/installation or replacement.
  • Combination switch assembly removal/installation or replacement.
  • LKAS control unit replacement.
  • Wheel alignment adjustment (the work which needs the attachment and removal/installation of the suspension component and the adjustment of toe).
