1. Install the starter in the reverse order of removal. Make sure the crimped side of the ring terminal (A) is facing out.
  1. Check the starter through bolt for tightness.
    Specified Torque:
    11 N·m (1.1 kgf·m, 8.0 lbf·ft)

  1. Install the bolt securing the harness holder.
  1. Install the dipstick/tube assembly with new O-ring.
  1. Install the expansion tank.
  1. Install the engine cover.
  1. Connect the positive cable to the battery first, then connect the negative cable.
  1. Start the engine to make sure the starter works properly.
  1. Enter the anti-theft code for the radio and the navigation system, then enter the customer's radio station presets.
  1. Set the clock.
  1. Do the power window control unit reset procedure.
