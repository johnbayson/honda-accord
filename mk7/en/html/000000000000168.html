How to Rewrite the ODO Data
NOTE:
  • To rewrite the ODO data, follow the new unit installation steps after gauge control module replacement.
  • If the HDS retrieves the ODO data from the gauge assembly, the ODO value on the ODO/TRIP display will appear as ''———'', maiking the ODO function unusable, and the HDS will be unable to retrieve additional ODO values.
  • Rewriting is nor possible on a gauge control module that does not communicate.
  • Obtain a new gauge control module, before staring the rewriting process.
  1. Before replacing the gauge control module, connect the HDS to the data link connector.
  1. Select the ''BODY ELECTRICAL SYSTEM'' from the HDS menu display.
  1. Select ''ODO REWRITE'' from the HDS menu display, and follow the instructions on the menu display to retrieve the ODO.
  1. Replace the gauge control module.
  1. Follow the instructions on the menu display to write the new ODO value to the new gauge control module.
