A/T Mainshaft Speed Sensor Replacement
  1. Remove the intake air duct and air cleaner housing.
  1. Disconnect the input shaft (mainshaft) speed sensor connector, and remove the input shaft (mainshaft) speed sensor.
  1. Install the new O-ring (A) on the new input shaft (mainshaft) speed sensor (B), then install the input shaft (mainshaft) speed sensor in the transmission housing.
  1. Check the connector for rust, dirt, or oil, then connect the connector securely.
  1. Install the intake air duct and air cleaner housing.
