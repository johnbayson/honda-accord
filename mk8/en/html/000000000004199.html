Special Tools Required

Sub hanger stay
 07MAK-PY30100

Engine tilt hanger set
 07KAK-SJ40101

Engine hanger stay
 07KAK-SJ40200
These tools use only for except EU models.
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Attach the engine tilt hanger set (Commercially available tool for EU models) to the engine as shown. Install the engine into position in the vehicle.
    NOTE: Reinstall the mounting bolts and nuts in the sequence given in the following steps. Failure to follow this sequence may cause excessive noise and vibration, and reduce engine mount life.
  1. Install the rear engine mount (A), then tighten the new rear mount mounting bolts (B).
  1. Loosely tighten the new rear mounting bolt (C).
  1. Tighten the front engine mount bolt (A), then install the front engine mount stop (B), vacuum line mounting bolt (C), and the vacuum hose (D).
  1. Tighten the rear mount mounting bolt.
  1. Raise the vehicle on the lift to full height.
  1. Install the transmission lower mount.
  1. Lower the vehicle on the lift.
  1. Remove the engine tilt hanger set from the engine.
M/T model
A/T model
  1. Remove the sub hanger stay (Commercially available tool for EU models).
  1. Install the side engine mount bracket (A), and tighten the new bolt and nut (B), then tighten the bolt (C).
  1. Install the ground cable (D).
  1. Install the transmission mount (A), then tighten the new transmission mount mounting bolts in the numbered sequence shown.
  1. Install the ground cables (B).
  1. Raise the vehicle on the lift to full height.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and the intermediate shaft.
  1. Connect the lower arms to the knuckles.
  1. Install the damper fork.
  1. A/T model: Install the shift cable.
M/T model
A/T model
  1. Install exhaust pipe A. Use new gaskets (B) and new self locking nuts (C).
  1. Install the splash shield.
  1. Install the front wheels.
  1. Lower the vehicle on the lift.
  1. A/T model: Install the automatic transmission fluid (ATF) cooler hoses (A) to the transmission, and secure the hoses with the clips (B).
  1. Install the upper radiator hose (A), lower radiator hose (B), and heater hoses (C).
  1. With A/C: Install the A/C compressor (A).
  1. With A/C: Connect the A/C compressor clutch connector (B), then install the harness clamps (C).
  1. With A/C: Install the drive belt.
  1. M/T model: Install the clutch slave cylinder, and the clutch line bracket mounting bolt.
  1. M/T model: Install the shift cable.
  1. Install the evaporative emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Connect the fuel hose (A), then install the quick-connect fitting cover (B).
  1. Install the intake air duct.
  1. Install the battery base (A), then install the harness clamp (B) and intake air duct mounting bolt (C).
  1. Connect the engine wire harness connector (A).
  1. Install the engine wire harness connector (B) to the bracket, then connect the connector.
  1. Install the harness clamps (C).
  1. Install the battery cables (D) to the under-hood fuse/relay box.
  1. Install the engine control module (ECM)/powertrain control module (PCM) bracket assembly (A).
  1. Connect the ECM/PCM connectors (B).
  1. Install the harness clamps (C).
  1. Install the strut brace.
  1. Install the air cleaner housing assembly.
  1. Do the battery installation procedure.
  1. A/T model: Move the shift lever to each gear, and verify that the A/T gear position indicator follows the transmission range switch.
  1. M/T model: Check that the transmission shifts into all gear smoothly.
  1. Inspect for fuel leaks. Turn the ignition switch to ON (II) (do not operate the starter) so the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid:
    • Manual transmission
    • Automatic transmission
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the ECM/PCM reset procedure.
  1. Do the ECM/PCM idle learn procedure.
  1. Do the crankshaft position (CKP) pattern clear/CKP pattern learn procedure.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
