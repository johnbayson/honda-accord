Power Steering Assist Check
NOTE: This test should be done with original equipment tyres and wheels at the correct tyre pressure.
  1. Start the engine, and let it idle.
  1. Attach a commercially available spring scale to the steering wheel. With the engine idling and the vehicle on a clean, dry floor, pull the scale as shown, and read it as soon as the tyres begin to turn.
  1. If the scale reads no more than 29 N (3.0 kgf, 6.6 lbf), the power assist is OK. If it reads more, check these items:
    • Steering linkage
    • Rack guide adjustment
    • EPS system
