Rewriting the ODO Data and Transferring the Service Reminder System Data to a New Gauge Control Module
NOTE:
  • Obtain a new gauge control module before starting the rewriting process.
  • Rewriting is not possible on a gauge control module that will not communicate with the HDS.
  • Make sure that the HDS shows the correct VIN for the vehicle you are working on.
  • Once you have started this procedure, you must complete it before removing the HDS from the DLC.
  • Connect a battery jumper box (not a battery charger) to insure that correct battery voltage will be maintained.
Rewriting and Transferring
  1. Before replacing the gauge control module, connect the HDS.
  1. Select GAUGES from the BODY ELECTRICAL system select menu with the HDS.
  1. Select Gauge Control Module Replacement (ODO Rewrite) from the ADJUSTMENT menu, and follow the instructions on the display to retrieve the ODO value and the Service Reminder System Information.
  1. Replace the gauge control module.
  1. Follow the instructions on the display to write the new ODO value and Service Reminder System Data to the new gauge control module. If the data transfer fails, refer to the instructions below to release the locked ODO value.
Release Locked odometer mileage to the original gauge control module.
If, after you attempt to transfer mileage, the odometer display has dashes (−−−), is garbled, or shows an incorrect value, the original gauge control module needs to be unlocked and restored to its original state:
  1. Confirm that you have the latest HDS version of software.
  1. Make sure that the HDS shows the correct VIN for the vehicle you are working on.
  1. With the ignition switch in LOCK (0), reconnect the original gauge control module.
  1. Completely re-boot the HDS.
  1. Clear any stored DTCs.
  1. Navigate to Body Electric/Gauges/Adjustment/Instrument Panel Replacement.
  1. Select 3. Releasing Locked ODO Value.
  1. Follow the prompts and the odometer mileage will be restored.
  1. Start over and make sure the screen prompts are followed.
