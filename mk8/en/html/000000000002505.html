A/T Intermediary Shaft 4th Gear Installation
Special Tools Required

Driver handle, 40 mm I.D.
 07746-0030100
  1. Wrap the shaft splines with tape, and install new O-rings in the 3rd clutch O-ring grooves, then remove the tape.
  1. Install the 3rd clutch, 53 mm splined washer, the thrust needle bearing, the needle bearing, 3rd gear, the thrust needle bearing on the intermediary shaft.
  1. Install 4th gear (A) using the 40 mm driver handle and a press.
  1. Install the 19.8 mm cotters, the cotter retainer, and the snap ring.
  1. Install the 26.5 mm washer on the top of the intermediary shaft.
