A/T Interlock System - Shift Lock Release, Release Spring, and Release Extension Replacement
NOTE: This replacement applies for the RHD model.
  1. Remove the shift lever assembly.
  1. Remove the shift lever knob.
  1. Remove the shift lever boot and the A/T gear position indicator panel base.
  1. Release the lock (A) of the shift lock release extension (B), and remove the shift lock release extension.
  1. Push the stop lever (C) of the shift lock release (D), and remove the shift lock release.
  1. Remove the shift lock release spring (A) from shift lock the release (B).
  1. Replace the shift lock release, the release spring, or the release extension.
  1. Install the shift lock release spring in the shift lock release.
  1. Install the shift lock release (A) while aligning the tab (B) with the tab slot (C), while hooking the release spring end (D) in the slot (E), and set the stop lever (F) on the stop (G).
  1. Install the shift lock release extension (H).
  1. Install the shift lever boot and the A/T gear position indicator panel base.
  1. Install the shift lever knob.
  1. Install the shift lever assembly.
