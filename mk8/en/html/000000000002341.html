Special Tools Required

Engine tilt hanger set
 07KAK-SJ40101

Engine hanger stay
 07KAK-SJ40200
These tools use only for except EU models
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Attach the engine tilt hanger set and the engine hanger stay (Commercially available tool for except EU models) to the engine as shown. Install the engine into position in the vehicle.
    NOTE: Reinstall the mounting bolts and nuts in the sequence given in the following steps. Failure to follow this sequence may cause excessive noise and vibration, and reduce engine mount life.
  1. Install the transmission mount (A), then tighten the new transmission mount mounting bolts (B).
  1. Loosely tighten the new transmission mount bracket mounting bolt and nuts (C), then install the ground cables (D).
  1. Install the side engine mount upper bracket (A), then tighten the new bolts and nut (B), and tighten the bolt (C).
  1. Tighten the transmission mount bracket mounting bolts and nut.
  1. Remove the engine tilt hanger set and the engine hanger stay from the engine.
  1. Raise the vehicle on the lift to full height.
  1. Install the lower torque rod, then tighten the new lower torque rod mounting bolts in the numbered sequence shown.
  1. Lower the vehicle on the lift.
  1. Install the upper torque rod (A), the torque rod stiffener (B), and the ground cable (C), then tighten the new torque rod stiffener mounting bolts (D).
    NOTE: The up mark (E) that installs the torque rod is upturned.
  1. Tighten the new upper torque rod mounting bolts in the numbered sequence shown.
  1. Raise the vehicle on the lift to full height.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and the intermediate shaft.
  1. Install exhaust pipe A. Use new gaskets (B) and new self locking nuts (C).
  1. Install the splash shield.
  1. Install the front wheels.
  1. Lower the vehicle on the lift.
  1. Install the A/C compressor (A), then connect the A/C compressor clutch connector (B).
  1. Install the water bypass hose (A), then install the two bolts (B).
  1. Connect the water bypass hoses, then install the expansion tank.
  1. Install the two bolts securing the expansion tank.
  1. Install the radiator.
  1. Install the shift cable.
  1. Install the ATF cooler hoses.
  1. Install the heater hoses.
  1. Install the intercooler outlet hose (A), turbocharger outlet hose (B), and water bypass hose (C).
  1. Install the fuel hose (A) with new O-ring (B).
  1. Install the fuel return hose (C) and brake booster vacuum hose (D).
  1. Install the battery base (A).
  1. Install the air cleaner housing bracket (A).
  1. Install the under-hood fuse/relay box (B), then install the battery cable (C) to the under-hood fuse/relay box.
  1. Install the harness holder (A) and harness clamps (B), then connect the engine wire harness connector (C) and the transmission control module (TCM) connector (D).
  1. Install the engine control module (ECM)/bracket assembly (A), then install the ground cable (B) and harness clamp (C).
  1. Install the intake air duct.
  1. Connect the ECM connectors (A), then install the connector cover (B) and harness clamps (C).
  1. Install the engine cover.
  1. Install the drive belt.
  1. Install the strut brace.
  1. Install the air cleaner housing assembly.
  1. Do the battery installation procedure.
  1. Move the shift lever to each gear, and verify that the A/T gear position indicator follows the transmission range switch.
  1. Do the fuel system bleed procedure.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid.
  1. Refill the expansion tank with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the ECM reset procedure.
  1. Inspect the idle speed.
