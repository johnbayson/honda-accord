LKAS System Troubleshooting
NOTE
  • Retrieve DTCs, using the HDS.
  • The LKAS system consists of the LKAS control unit, ACC unit, a LKAS camera and the EPS control unit. Depending on the trouble that has been occurred, each unit or the camera individually writes in DTCs. Therefore, when a trouble occurs, determine which unit writes these DTCs at first, then starts diagnosing the trouble from the units or the LKAS camera the DTCs are coming from.
  • Information marked with an asterisk ( *1 ) applies to LHD model, an asterisk ( *2 ) applies to RHD model.
