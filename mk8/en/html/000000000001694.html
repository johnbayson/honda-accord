M/T Mainshaft Reassembly

Exploded View - K24Z3 Engine Model
Special Tools Required

Inner driver handle, 40 mm
 07746-0030100

Inner bearing driver attachment, 30 mm
  07746-0030300
K24Z3 Engine Model
NOTE: Refer to the Exploded View, as needed, during this procedure.
  1. Clean all the parts in solvent, dry them, and apply MTF to all contact surfaces.
  1. Install the needle bearing (A) and the 3rd gear (B) onto the mainshaft (C).
  1. Install the double cone synchro assembly (A) with the synchro spring (B) by aligning the synchro cone fingers (C) with the holes (D) in the 3rd gear.
  1. Install the 3rd/4th synchro hub (A) by aligning the synchro ring fingers (B) with the grooves (C) in the 3rd/4th synchro hub.
    NOTE: Make sure the installing direction of the 3rd/4th synchro hub.
  1. Press in the 3rd/4th synchro hub (A) using the 40 mm inner driver handle (B).
  1. Install the 3rd/4th synchro sleeve (A) by aligning the stops (B) of the 3rd/4th synchro sleeve and the 3rd/4th synchro hub. After installing, check the operation of the 3rd/4th synchro hub set.
  1. Install the double cone synchro assembly (A) with the synchro spring (B) by aligning the synchro ring fingers (C) with the grooves (D) in the 3rd/4th synchro hub.
  1. Install 4th gear (A) by aligning the synchro cone fingers (B) with the holes (C) in 4th gear.
  1. Install the 4th/5th gear distance collar (A) with the needle bearings (B) and 5th gear (C).
  1. Install the synchro ring (D) with the synchro spring (E) onto the 5th gear.
  1. Install the 5th/6th synchro hub (A) by aligning the synchro ring fingers (B) with the grooves (C) in the 5th/6th synchro hub.
  1. Press in the 5th/6th synchro hub (A) using the 40 mm inner driver handle (B) and 30 mm inner bearing driver attachment (C).
  1. Install the 5th/6th synchro sleeve (A) by aligning the slots of the 5th/6th synchro sleeve and the 5th/6th synchro hub (B). After installing, check the operation of the 5th/6th synchro hub set.
    NOTE: The slots layout is inequable. Set correctly them.
  1. Install the synchro ring (A) with the synchro spring (B) by aligning the synchro ring fingers (C) with the grooves (D) in the 5th/6th synchro hub.
  1. Install the 6th gear distance collar (E) with the needle bearing (F), and 6th gear (G).
  1. Press in the new angular ball bearing (A) using the 40 mm inner driver handle (B), 30 mm inner bearing driver attachment (C), and a press (D).
