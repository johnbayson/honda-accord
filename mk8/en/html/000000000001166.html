Accelerator Pedal Module Removal and Installation
Removal
  1. Disconnect the accelerator pedal module connector (A).
  1. Remove the clip (B).
  1. Using flat-tip screwdriver (C), push the lock (D), then remove the accelerator pedal module (E).
    NOTE:
    • APP sensor is not available separately. Do not disassemble the accelerator pedal module.
    • If the pedal stop (F) is damaged, pull back the carpet, remove the nut (G), and replace the pedal stop.
Installation
  1. Install the accelerator pedal module (A), then slide it until lock with a clicking sound.
  1. Install a new clip (A).
    NOTE:
    • Make sure that the accelerator pedal module and the clip are secure.
    • The clip must be replaced with a new one.
  1. A/T: Do the accelerator pedal kickdown click system learn procedure.
