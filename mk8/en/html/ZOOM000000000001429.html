Keyless/Power Door Locks/Security System Symptom Troubleshooting Information
Power Door Locks/Keyless
  1. Check for B-CAN DTCs. If any B-CAN DTCs are indicated, troubleshoot and resolve them first.
  1. If the door lock system and the keyless operation do not work, troubleshoot the door locks first.
    NOTE: The system does not function when the ignition switch is ON (II).
    No.
    Symptom
    Check Items
    1
    All the doors will not lock or unlock. *1
    • Poor ground (G501, G502, G505, G601, G651)
    • Driver's door key cylinder switch test
    • Door switch test (check the door switch ON/OFF information with the HDS)
    • Door lock switch test (check the door switch ON/OFF information with the HDS)
    • Also, check for continuity of the UART line between the door multiplex control unit 37P connector No. 27 terminal and the front passenger's power window switch 37P connector No. 29 terminal
    2
    Driver's, left rear doors and fuel fill door lock actuator will not lock or unlock.
    • Poor ground (G501, G502, G505, G601)
    • Blown No. 24 (20 A) fuse in the driver's under-dash fuse/relay box
    • Blown No. 25 (15 A) fuse in the driver's under-dash fuse/relay box
    • Driver's MICU input test
    • Door multiplex control unit input test
    3
    Front passenger's and right rear doors will not lock or unlock.
    • Poor ground (G501, G502, G505, G651)
    • Blown No. 1 (AS F/B) (40 A) *2 or (60 A) *3 fuse in the under-hood fuse/relay box
    • Blown No. 10 (10 A) fuse in the passenger's under-dash fuse/relay box
    • Blown No. 13 (20 A) fuse in the passenger's under-dash fuse/relay box
    • Passenger's MICU input test
    • Door multiplex control unit input test
    4
    Keyless operation does not work (LOCK, UNLOCK).
    Symptom troubleshooting
    5
    Doors will not unlock with the transmitter, but will unlock with the door lock switch and the door key cylinder switch.
    • Symptom troubleshooting
    • Door lock switch test (check the door switch ON/OFF information with the HDS)
    6
    Doors will not lock with the transmitter, but will lock with the door lock switch and the door key cylinder switch.
    • Symptom troubleshooting
    • Door lock switch test (check the door switch ON/OFF information with the HDS)
    7
    Doors automatically relock 30 seconds after being unlocked with the transmitter even though a door has been opened.
    • Symptom troubleshooting
    • Door switch (check the door switch ON/OFF information with the HDS)
    8
    Only driver's door will unlock or door locks relock immediately after unlocking with the remote.
    Driver's door lock knob switch test
    9
    Keyless operation will work even though the ignition key is in the ignition switch.
    Ignition key switch test

    *1:
    If only one door is not working properly, check that door's lock actuator first, then check the other items listed in this table.
