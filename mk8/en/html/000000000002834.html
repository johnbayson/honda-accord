A/T 1st, 2nd, and 3rd Clutch Reassembly
Special Tools Required

Clutch spring compressor set
 07LAE-PX40000

Clutch spring compressor attachment
 07LAE-PX40100

Clutch spring compressor attachment
 07HAE-PL50100

Clutch spring compressor bolt set 10
 07GAE-PG40200

Snap ring pliers
 07LGC-0010100
NOTE:
  • Hold the spring compressor attachment in a vise with soft jaws. Be careful not to damage the clutch drum.
  • Insert the spring retainer can be adjusted (center of tolerance) to prevent damaging the spring retainer oil seal.
  1. Soak the clutch discs thoroughly in ATF for at least 30 minutes.
  1. Install a new O-ring (A) in the 1st, 2nd, and 3rd clutch pistons (B), and install a new O-ring (C) on each clutch drum (D).
  1. Install the clutch piston (A) in the clutch drum (B). Apply pressure and rotate to ensure proper seating. Lubricate the piston O-ring with ATF before installing. Do not pinch the O-ring by installing the piston with too much force.
  1. Set the return spring (A) and the spring retainer (B) in the clutch drum (C), then position the snap ring (D) on the spring retainer.
  1. Install the clutch spring compressor attachments and clutch spring compressor bolt set 10.
  1. Set the clutch spring compressor attachment (A) on the spring retainer (B) of the 1st, 2nd, and 3rd clutches in such a way that it compresses the clutch return spring (C).
    NOTE: Coat the circumference of the spring retainer and the areas where the spring retainer contacts the clutch piston with ATF before installation.
  1. Compress the return spring carefully until the snap ring can be installed. Check the spring retainer (A) is properly installed on the clutch hub (B). If improperly installed, change the position of the spring compressor attachment and the spring retainer.
    NOTE: Insert the spring retainer so it can be adjusted (center of tolerance) to prevent damaging the spring retainer oil seal.
  1. Install the snap ring in the clutch hub using the snap ring pliers.
  1. Remove the clutch spring compressor attachments and clutch spring compressor bolt set 10.
  1. Make sure the oil seal of the spring retainer (A) is properly installed on the clutch piston (B). If the oil seal was damaged or cracked, replace the spring retainer.
  1. For R20A3 engine model 1st clutch: Install the waved spring (A) in the 1st clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (5) and the clutch discs (D) (5), then install the clutch end-plate (E) with the flat side toward the top disc.
  1. For R20A3 engine model 2nd clutch: Install the waved spring (A) in the 2nd clutch drum (B). Install the 4.0 mm-thick plate (C) with the shear droop side (D) toward the inside of the drum, the clutch disc (E), and the 2.0 mm-thick clutch flat-plate (F). Starting with the clutch disc, alternately install the clutch discs (G) (4) and the wave-plates (H) (3), then install the clutch end-plate (I) with the flat side toward the top disc.
  1. For R20A3 engine model 3rd clutch: Install the waved spring (A) in the 3rd clutch drum (B). Install the clutch flat-plate (C), the clutch disc (D), and the clutch flat-plate (E). Starting with the clutch disc, alternately install the clutch discs (F) (4) and the wave-plates (G) (3), then install the clutch end-plate (H) with the flat side toward the top disc.
  1. For K24Z3 engine model 1st clutch: Install the waved spring (A) in the 1st clutch drum (B). Install the clutch flat-plate (C) (1), then starting with the clutch disc, alternately install the clutch discs (D) (5) and the clutch wave-plates (E) (4). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. For K24Z3 engine model 2nd clutch: Install the waved spring (A) in the 2nd clutch drum (B). Install the clutch flat-plate (C) (1), then starting with the clutch disc, alternately install the clutch discs (D) (6) and the clutch wave-plates (E) (5). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. For K24Z3 engine model 3rd clutch: Install the waved spring (A) in the 3rd clutch drum (B). Install the clutch flat-plate (C) (1), then starting with the clutch disc, alternately install the clutch discs (D) (6) and the clutch wave-plates (E) (5). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. Install the snap ring using a screwdriver to secure the clutch end-plate.
  1. Check that the clutch piston moves by applying air pressure into fluid passage.
