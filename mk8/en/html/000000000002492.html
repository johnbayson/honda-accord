  1. Remove the bolt, then remove the lock washer (A) and the bearing set plate (B).
  1. Remove the secondary shaft bearing from the torque converter housing using the adjustable bearing remover set.
  1. Remove the ATF guide collar from the torque converter housing.
  1. Remove the 29 x 39 x 9.5 mm secondary shaft bearing (A) from the torque converter housing using the 30 mm bearing remover shaft set, the remover handle, and the weight remover.
  1. Install the 29 x 39 x 9.5 mm bearing into the torque converter housing using the driver handle and the 37 x 40 mm attachment.
  1. Install new O-rings (A) on the ATF guide collar (B), then install the guide collar in the torque converter housing.
  1. Install the bearing in the direction shown in the torque converter housing using the driver handle and the 62 x 68 mm attachment.
  1. Check that the bearing groove aligns with the set plate mounting surface, then install the set plate (A) by aligning it with the bearing groove.
  1. Install a new lock washer (B) and the bolt, then bend the lock tab of the lock washer against the bolt head.
