A/T Intermediary Shaft 3rd Gear Axial Clearance Inspection
Special Tools Required

Driver handle, 40 mm I.D.
 07746-0030100
  1. Install the 3rd clutch (A), the selected 53 mm splined washer (B), the 30 x 52 x 3 mm thrust needle bearing (C), the needle bearing (D), intermediary shaft 3rd gear (E), and the 30 x 52 x 3 mm thrust needle bearing (F) on the intermediary shaft (G). Do not install the O-rings during inspection.
  1. Install intermediary shaft 4th gear (H) using the 40 mm driver handle and a press.
  1. Install the 19.8 mm cotters (I), the cotter retainer (J), and the snap ring (K).
  1. Set the dial indicator (A) on intermediary shaft 3rd gear (B).
  1. Measure the intermediary shaft 3rd gear axial clearance in at least three places while moving intermediary shaft 3rd gear. Use the average as the actual clearance.
    Standard: 0.005−0.045 mm (0.0002−0.0018 in.)

  1. If the clearance is out of the standard, remove the 53 mm splined washer and measure its thickness.
  1. Select and install a new 53 mm splined washer, then recheck the clearance.
