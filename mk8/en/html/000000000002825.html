A/T Clutch Clearance Inspection
Special Tools Required

Clutch compressor attachment
 07ZAE-PRP0100
  1. Inspect the clutch piston, the discs, the plates, and the end-plate for wear and damage, and inspect the clutch wave-plate phase difference, if necessary.
  1. Install the clutch piston in the clutch drum. Do not install the O-rings during inspection.
  1. For R20A3 engine model 1st clutch: Install the waved spring (A) in the 1st clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (5) and the clutch discs (D) (5), then install the clutch end-plate (E) with the flat side toward the top disc.
  1. For R20A3 engine model 2nd clutch: Install the waved spring (A) in the 2nd clutch drum (B). Install the 4.0 mm-thick plate (C) with the shear droop side (D) toward the inside of the drum, the clutch disc (E), and the 2.0 mm-thick clutch flat-plate (F). Starting with the clutch disc, alternately install the clutch discs (G) (4) and the wave-plates (H) (3), then install the clutch end-plate (I) with the flat side toward the top disc.
  1. For R20A3 engine model 3rd clutch: Install the waved spring (A) in the 3rd clutch drum (B). Install the clutch flat-plate (C) (1), the clutch disc (D), and the clutch flat-plate (E). Starting with the clutch disc, alternately install the clutch discs (F) (4) and the wave-plates (G) (3), then install the clutch end-plate (H) with the flat side toward the top disc.
  1. For K24Z3 engine model 1st clutch: Install the waved spring (A) in the 1st clutch drum (B). Install the clutch flat-plate (C) (1), then starting with the clutch disc, alternately install the clutch discs (D) (5) and the clutch wave-plates (E) (4). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. For K24Z3 engine model 2nd clutch: Install the waved spring (A) in the 2nd clutch drum (B). Install the clutch flat-plate (C) (1), then starting with the clutch disc, alternately install the clutch discs (D) (6) and the clutch wave-plates (E) (5). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. For K24Z3 engine model 3rd clutch: Install the waved spring (A) in the 3rd clutch drum (B). Install the clutch flat-plate (C) (1), then starting with the clutch disc, alternately install the clutch discs (D) (6) and the clutch wave-plates (E) (5). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. Install the waved spring (A) in the 4th clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (4) and the clutch discs (D) (4). Install the clutch end-plate (E) with the flat side toward the top disc.
  1. Install the waved spring (A) in the 5th clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (4) and the clutch discs (D) (4). Install the clutch end-plate (E) with the flat side toward the top disc.
  1. Install the snap ring using a screwdriver to secure the clutch end-plate.
  1. Set a dial indicator (A) on the clutch end-plate (B).
  1. Zero the dial indicator with the clutch end-plate lifted up to the snap ring (C).
  1. Release the clutch end-plate to lower the clutch end-plate, then put the clutch compressor attachment on the clutch end-plate (A).
  1. Press the clutch compressor attachment down with 147 N (15 kgf, 33 lbf) (B) using a force gauge, and read the dial indicator (C). The dial indicator reads the clearance (D) between the clutch end-plate and the top disc (E). Take measurements in at least three places, and use the average as the actual clearance.
