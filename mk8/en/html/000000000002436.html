A/T Assembly Reassembly

Exploded View
Special Tools Required

Mainshaft holder
 07PAB-0010001

Snap ring pliers
 07LGC-0010100
  1. Make sure that the ATF magnet is cleaned and installed in the torque converter housing. Clean and install the ATF magnet, if necessary.
  1. Install the main separator plate and the two dowel pins on the torque converter housing. Then install the ATF pump drive gear (A), the driven gear (B), and the ATF pump driven gear shaft (C). Install the ATF pump driven gear with its grooved and chamfered side facing down.
  1. Install the torque converter check valve spring and the valve in the torque converter housing.
  1. Install the main valve body (seven 6 mm bolts and three 8 mm bolts). Make sure the ATF pump drive gear (A) rotates smoothly in the normal operating direction, and the ATF pump driven gear shaft (B) moves smoothly in the axial and normal operating direction.
  1. If the ATF pump drive gear and the ATF pump driven gear shaft do not move smoothly, loosen the main valve body bolts. Realign the ATF pump driven gear shaft, and retighten the bolts to the specified torques, then recheck. Failure to align the ATF pump driven gear shaft correctly will result in a seized ATF pump drive gear or ATF pump driven gear shaft.
  1. Install the lubrication check valve in the main valve body, in the direction shown in the exploded view.
  1. Install the secondary separator plate and the two dowel pins on the main valve body, and install the secondary valve body (one bolt).
  1. Install the two check balls in the secondary valve body.
  1. Install the accumulator separator plate and the two dowel pins on the secondary valve body.
  1. Position the detent arm on the accumulator separator plate, and install the detent arm shaft into the detent arm through the separator plates to the main valve body.
  1. Install the 8 x 62 mm ATF feed pipe in the main valve body, and install the accumulator body (11 bolts).
  1. Install the regulator separator plate and the two dowel pins on the main valve body.
  1. Install the stator shaft with a new O-ring, and install the regulator valve body (seven bolts).
  1. Install the stator shaft stop in the main valve body.
  1. Install the ATF strainer with a new O-ring (one bolt).
  1. Install the ATF passage pipe (one bolt) in the torque converter housing.
  1. Install the intermediary shaft (A) into the main valve body, and install the 26.5 mm washer (B) on the top of the intermediary shaft.
  1. Install the differential assembly (C) in the torque converter housing.
  1. Install the baffle plate (D), and make sure if the differential is clear of the baffle plate.
  1. Install the baffle plate (E), and make sure if the intermediary shaft is clear of the baffle plate.
  1. Install the control shaft and the park lever link (F).
  1. Assemble the mainshaft, the countershaft, and the secondary shaft.
  1. Install the thrust needle bearing (A) on the secondary shaft bearing in the torque converter housing.
  1. Turn the shift fork shaft (B) so the large chamfered hole is facing the fork bolt hole of the shift fork.
  1. Engage the shift fork (C) with the reverse selector on the countershaft, and join the mainshaft (D), countershaft (E), and the secondary shaft (F), then install them in the torque converter housing and the shift fork on the shift fork shaft.
  1. Secure the shift fork (A) to the shift fork shaft with the lock bolt and a new lock washer (B), then bend the lock tab of the lock washer against the bolt head.
  1. Install the park pawl shaft (A), the pawl spring (B), the park pawl (C), and the park pawl stop (D).
  1. Align the control lever pin (A) with the manual valve guide (B).
  1. Hook the detent spring (A) to the detent arm (B).
  1. Install the 8 x 85 mm ATF feed pipe (A), the 8 x 151.5 mm pipes (B), and the 8 x 40 mm pipe (C) in the accumulator body.
  1. Install the 8 x 57.5 mm ATF feed pipe (A) and the 10 x 123 mm pipe (B) in the torque converter housing.
  1. Install the two dowel pins (A) and a new gasket (B) on the torque converter housing (C).
  1. Place the transmission housing (D) on the torque converter housing.
  1. Install the ATF warmer bracket (E).
  1. Install the ATF cooler pipe mounting bracket (F) and the harness brackets (G).
  1. Wrap a screwdriver tip with tape to prevent damage to the reverse idler gear teeth. Engage the reverse idler gear with the reverse gears by rotating the idler gear using a screwdriver.
  1. While expanding the snap ring of the countershaft bearing using snap ring pliers, install the transmission housing onto the bearing part-way. Then release snap ring pliers, and push down on the transmission housing until it bottoms and snap ring snaps into place in the transmission housing snap ring groove.
  1. Install the transmission housing mounting bolts (A) (20 bolts) along with the transmission hanger (B) and the ground terminal bracket (C), and tighten the bolts in at least two steps in a criss-cross pattern to 44 N·m (4.5 kgf·m, 33 lbf·ft).
  1. Install and tighten the two mounting bolts (D) to 39 N·m (4.0 kgf·m, 29 lbf·ft). Keep the mounting bolts free of grease or oil.
  1. Install and tighten the 12 x 1.25 mm bolt (E) to 64 N·m (6.5 kgf·m, 47 lbf·ft).
  1. Install the mainshaft holder onto the mainshaft.
  1. Install a new lock washer (A) with the marked side (B) up over the mainshaft (C), and apply ATF to surfaces of the lock washer and the old locknut (D).
  1. Install the old locknut, and tighten it to seat the lock washer to 178 N·m (18.2 kgf·m, 132 lbf·ft), then remove the old locknut.
  1. Install a new locknut, and tighten it to 178 N·m (18.2 kgf·m, 132 lbf·ft), then stake the locknut to a depth (A) of 0.7−1.3 mm (0.03−0.05 in.) using a 3.5 mm punch (B).
  1. Install the end cover (A), the dowel pins (B), and a new O-ring (C).
  1. Install the snap ring cap (D) and a new O-ring (E).
  1. Apply thread lock sealant to the threads of the sealing plug (F), and install the sealing plug and a new sealing washer (G).
  1. Install a new O-ring (E) on the solenoid wire harness connector (F).
  1. Route the solenoid wire harness through the transmission housing, and install the solenoid wire harness connector.
  1. Connect the shift solenoid wire harness terminals to the shift solenoid valves:
    • YEL wire connector to shift solenoid valve A.
    • RED wire connector to shift solenoid valve B.
    • GRN wire connector to shift solenoid valve C.
    • ORN wire connector to shift solenoid valve D.
  1. Secure the solenoid wire harness connector with the bolt on the transmission housing.
  1. Install the solenoid valve cover (A) with the two dowel pins (B) and a new gasket (C), and secure it with the seven bolts.
  1. Place a new gasket (A) on the solenoid valve cover, then install the 8 x 105.8 mm ATF feed pipe (B) and the 8 x 58.3 mm pipes (C) with the filter end in the transmission housing.
  1. Install new O-rings (D) over the ATF feed pipes, and install A/T clutch pressure control solenoid valves A and B (E).
  1. Install the 8 x 53 mm ATF joint pipe (A) with the filter end into its mounting hole (B).
  1. Check the height (A) of the 8 x 53 mm ATF joint pipe (B) between the top (C) of the pipe and the solenoid valve body mounting surface (D). The height is about 7 mm (0.3 in.). If the height is over 7 mm (0.3 in.), install the pipe securely until it stops the accumulator body.
  1. Install a new gasket (A) on the transmission housing. Install the 8 x 36 mm ATF pipe (B) with the filter end in the transmission housing, and install the 8 x 25.2 mm ATF pipe (D).
  1. Install new O-rings (E) over the ATF joint pipes.
  1. Install A/T clutch pressure control solenoid valve C.
  1. With a 6.0 mm wrench, turn the selector control shaft fully counterclockwise (viewed from the shaft end) to the P position. Turn the selector control shaft back two click-stopped positions so that it is in the N position.
  1. Set the transmission range switch (A) to the N position. The transmission range switch clicks in the N position, and the selector control shaft hole (B) aligns with the N positioning line (C).
  1. Install the transmission range switch gently over the selector control shaft (D), and install the bolts loosely.
  1. Install a new lock washer (A) over the selector control shaft (B) by aligning the indicator tab (C) of the lock washer with the N positioning line (D) on the transmission range switch (E), and install the locknut (F).
  1. Push the locknut against the transmission housing to seat the range switch into the selector control shaft, and tighten the locknut (A) to 12 N·m (1.2 kgf·m, 8.7 lbf·ft) while holding the selector control shaft (B) with a 6 mm wrench (C), then bend the lock tabs (D) against the locknut.
  1. Tighten the bolt (E) securing the transmission range switch to 12 N·m (1.2 kgf·m, 8.7 lbf·ft).
  1. Install the control lever (A), spring washer (B), a new lock washer (C), and locknut (D) on the selector control shaft (E). Bend the lock tab of the lock washer against the locknut.
  1. Install a new O-ring (A) on the input shaft (mainshaft) speed sensor (B), and install the input shaft (mainshaft) speed sensor.
  1. Install a new O-ring (C) on the output shaft (countershaft) speed sensor (D), and install the output shaft (countershaft) speed sensor and the sensor washer (E).
  1. Install the 4th clutch transmission fluid pressure switch (A) with a new sealing washer (B). Tighten the switch by gripping the metal part, not the plastic part.
  1. Install the 3rd clutch transmission fluid pressure switch (A) with a new sealing washer (B). Tighten the switch by gripping the metal part, not the plastic part.
  1. Install the ATF cooler lines (C) with new sealing washers (D) and the line bolts (E).
  1. Secure the ATF cooler lines with the 6.0 mm bolts (F). Tighten the bolts to 12 N·m (1.2 kgf·m, 8.7 lbf·ft).
  1. Install the ATF temperature sensor (G) with a new O-ring (H). Install the harness clamp on the clamp bracket, and install the connector on the connector bracket.
  1. Install the 2nd clutch transmission fluid pressure switch (A) with a new sealing washer (B). Tighten the switch by gripping the metal part, not the plastic part.
  1. Install the ATF dipstick guide tube (A) with a new O-ring (B).
  1. Install the ATF dipstick (C) in the guide tube.
