  1. Soak the clutch discs thoroughly in ATF for a minimum of 30 minutes.
  1. Install new O-rings (A) on the 1st-hold clutch piston (B), and install the 1st-hold clutch piston in the 1st clutch piston (C).
  1. Install new O-rings (A) (B) on the 1st clutch piston (C), and install the 1st/1st-hold clutch piston in the 1st clutch drum (D) while applying pressure and rotating to ensure proper seating. Do not pinch the O-ring.
  1. Install the O-rings (A) on the 2nd, 3rd, 4th and 5th clutch piston (B).
  1. Install the clutch pistons in the clutch drums (C) while applying pressure and rotating to ensure proper seating. Do not pinch the O-ring.
  1. Set the return spring (A) and the spring retainer (B), and position the snap ring (C) on the spring retainer.
  1. Install the clutch spring compressor set.
  1. Be sure the clutch spring compressor set (A) is adjusted to have full contact with the spring retainer (B) on the 1st, 2nd, and 4th clutches.
  1. Check the placement of the clutch spring compressor set. If either end of the clutch spring compressor set is set over an area of the spring retainer that is unsupported by the return spring, the spring retainer may be damaged.
  1. Set the clutch spring compressor set (A) on the spring retainer (B) of the 3rd and 5th clutches so that it compresses the clutch return spring (C).
  1. Check the placement of the clutch spring compressor set. If either end of the clutch spring compressor set is not set over the clutch return spring end (A), the spring retainer may be damaged.
  1. Compress the return spring until the snap ring can be installed.
  1. Install the snap ring using snap ring pliers.
  1. Remove the clutch spring compressor set.
  1. Install the 1st-hold clutch flat-plate (A), alternately install the clutch discs (B) (3) and the 1st-hold clutch wave-plates (C) (2) in the 1st/1st-hold clutch drum (D), then install 1st-hold clutch plate B (E) with the flat side toward the top disc.
  1. Install the disc spring (F) on 1st-hold clutch plate B in the direction shown. Starting with the 1st clutch wave-plate, alternately install the clutch wave-plates (G) (5) and the 1st clutch discs (H) (5), then install the clutch end-plate (I) with the flat side toward the top disc.
  1. Starting with the 2nd clutch flat-plate, alternately install the clutch flat-plates (A) (4) and the clutch discs (B) (4) in the 2nd clutch drum (C), and alternately install the clutch wave-plates (D) (3) and the clutch discs (B) (3). Install the clutch end-plate (E) with the flat side toward the top disc.
  1. Starting with the 3rd clutch flat-plate, alternately install the flat-plates (A) (3) and discs (B) (3) in the 3rd clutch drum (C), and alternately install the wave-plates (D) (3) and discs (B) (3). Install the clutch end-plate (E) with the flat side down on the top disc.
  1. Install the disc spring (A) in the 4th clutch drum (B) in the direction shown. Starting with the 4th clutch flat-plate, alternately install the clutch flat-plates (C) (3) and the clutch discs (D) (3), and alternately install the clutch wave-plates (E) (2) and the clutch discs (D) (2). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. Install the disc spring (A) in the 5th clutch drum (B) in the direction shown. Starting with the 5th clutch flat-plate, alternately install the clutch flat-plates (C) (3) and the clutch discs (D) (3), and alternately install the clutch wave-plates (E) (2) and the clutch discs (D) (2). Install the clutch end-plate (F) with the flat side toward the top disc.
  1. Install the snap ring using a screwdriver to secure the clutch end-plate.
  1. Check that the clutch piston moves by applying air pressure into the fluid passage.
