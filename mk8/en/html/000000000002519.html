A/T Valve Body Valve Installation
  1. Coat all parts with ATF before assembly.
  1. Install the valves and springs in the sequence shown for the main valve body, the secondary valve body, the regulator valve body, and the accumulator body. Refer to the following valve cap illustrations, and install each valve cap so the end shown facing up will be facing the outside of the valve body, then secure the valve cap with the valve cap clip.
  1. Install the valve (A) and the valve spring (B) in the valve body. Push the valve spring in using a screwdriver, then install the spring seat (C).
