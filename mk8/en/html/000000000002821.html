A/T Countershaft 3rd Gear and Reverse Selector Hub Installation
Special Tools Required

Inner driver handle 40 mm
 07746-0030100
NOTE: Be sure install 2nd gear, 1st gear, 5th gear, 3rd gear, and the 35 x 47 x 7.8 mm collar in the direction show.
  1. Install 2nd gear, 1st gear, 5th gear, and the 37 x 41 x 54.3 mm collar on the countershaft.
  1. Slide 3rd gear (A) over the countershaft, and press it in place using the 40 mm inner driver handle and a press.
  1. Install the 31 mm cotters, the 35 x 47 x 7.8 mm collar, the set ring, the needle bearing, and 4th gear.
  1. Slide the reverse selector hub (A) over the countershaft, then press it in place using the 40 mm driver and a press.
    NOTE: Some reverse selector hubs are not press-fitted, and can be installed without using the 40 mm inner driver handle and a press.
