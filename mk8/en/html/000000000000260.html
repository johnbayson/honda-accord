Audio System Symptom Troubleshooting - Audio unit button does not work
Audio unit button does not work
NOTE: In order to troubleshoot the main power switch, go to Power switch will not turn on.
  1. Go to the audio button, knob, and remote switch detection mode in the audio self-diagnostic function. Operate all items in the appropriate switch list.
    Switch list
    With navigation:
    FM, AM, TA, VOL, , CD, AUX, 1-6, FOLDER, TUNE, SKIP, SCAN/A.SEL, MAP/GUIDE, CANCEL, MENU, AUDIO, , INFO, ENTER
    With premium:
    1-6, LOAD, EJECT, FM, AM, TA, CD/AUX, TITLE, CLOCK, SCAN, RTY/FOLDER, SKIP, A.SEL, RETURN, ENTER, MENU
    Without premium:
    1-6, EJECT, FM, AM, CD, TA, CD, AUX, TITLE, CLOCK, SCAN, PIY/FOLDER, A.SEL, SKIP, VOL, RETURN, ENTER, MENU

Are all the items in the appropriate switch list detected?
YES -
Operation is normal.■
NO -
Substitute a known-good audio unit with navigation system, without navigation system and recheck. If the symptom/indicated goes away, replace the original audio unit with navigation system, without navigation system. If the symptom is still present, substitute a known-good audio/navigation switch panel and recheck. If the symptom/indication goes away, replace the original audio/navigation switch panel. ■
