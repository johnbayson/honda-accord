A/T Input Shaft (Mainshaft) Speed Sensor Replacement
  1. Do the battery removal procedure.
  1. Remove the battery base.
  1. Disconnect the input shaft (mainshaft) speed sensor connector, and remove the input shaft (mainshaft) speed sensor (A).
  1. Install a new O-ring (B) on a new input shaft (mainshaft) speed sensor, then install the input shaft (mainshaft) speed sensor.
  1. Check the connector for rust, dirt, or oil, then connect the connector securely.
  1. Install the battery base.
  1. Do the battery installation procedure.
