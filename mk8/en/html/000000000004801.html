Special Tools Required

Engine tilt hanger set
 07KAK-SJ40101

Engine hanger stay
 07KAK-SJ40200

Sub hanger stay
 07MAK-PY30100
These tools use only for except EU models.
  1. Install the engine mount brackets and the accessory brackets and tighten their bolts to the specified torque.
  1. Attach the special tools (Commercially available tool for EU models) to the engine as shown. Install the engine into position in the vehicle.
    NOTE: Reinstall the mounting bolts and nuts in the sequence given in the following steps. Failure to follow this sequence may cause excessive noise and vibration, and reduce engine mount life.
  1. Install the transmission mount (A), then tighten the new transmission mount mounting bolts (B).
  1. Loosely tighten the new transmission mount bracket mounting bolt and nuts (C), then install the ground cable (D).
  1. Support the engine with a jack and a wood block under the oil pan.
  1. Remove the special tools (Commercially available tool for EU models) from the engine.
  1. Install the side engine mount upper bracket (A), then tighten the new bolts and nut (B), and tighten the bolt (C).
  1. Tighten the transmission mount bracket mounting bolts and nut.
  1. Raise the vehicle on the lift.
  1. Install the lower torque rod, then tighten the new lower torque rod mounting bolts in the numbered sequence shown.
  1. Lower the vehicle on the lift.
  1. Install the upper torque rod (A), the torque rod stiffener (B), and the ground cable (C), then tighten the new torque rod stiffener mounting bolts (D).
    NOTE: The mark (E) that is installed on the torque rod is upturned.
  1. Tighten the new upper torque rod mounting bolts in the numbered sequence shown.
  1. Raise the vehicle on the lift.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and the intermediate shaft.
  1. Connect the lower arms to the knuckles.
  1. Install the damper fork.
  1. Install exhaust pipe A. Use new gaskets (B) and new self locking nuts (C).
  1. Install the splash shield.
  1. Install the front wheels.
  1. Lower the vehicle on the lift.
  1. Install the A/C compressor (A), then connect the A/C compressor clutch connector (B).
  1. Install the expansion tank.
  1. Install the radiator.
  1. Connect the heater hoses.
  1. Connect the intercooler outlet hose (A), the turbocharger outlet hose (B), and the water bypass hose (C).
  1. Install the fuel hose (A) with new O-ring (B).
  1. Connect the fuel return hose (C) and the brake booster vacuum hose (D).
  1. Install the clutch slave cylinder, and the clutch line bracket mounting nut.
  1. Install the shift cable.
  1. Install the battery base (A), then install the harness clamp (B).
  1. Install the air cleaner housing bracket (A) and the harness clamps (B).
  1. Install the under-hood fuse/relay box (C), then install the battery cable (D) to the under-hood fuse/relay box.
  1. Install the harness holder (A) and the harness clamps (B), then connect the engine wire harness connector (C).
  1. Install the engine control module (ECM)/bracket assembly (A), then install the ground cable (B) and the harness clamp (C).
  1. Install the intake air duct.
  1. Connect the ECM connectors (A), then install the connector cover (B) and the harness clamps (C).
  1. Install the engine cover.
  1. Install the strut brace.
  1. Install the air cleaner housing assembly.
  1. Do the battery installation procedure.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid.
  1. Check that the transmission shifts into gear smoothly.
  1. Refill the expansion tank with engine coolant, and bleed the air from the cooling system with the heater valve open.
  1. Do the ECM reset procedure.
  1. Inspect the idle speed.
