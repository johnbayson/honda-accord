Power Window Control Unit Reset
Resetting the power windows are required when any of the following have occurred:
  • Power window regulator replacement or repair
  • Power window motor replacement or repair
  • Window run channel replacement
  • Power window master switch or front passenger's power window switch replacement *
  • Left or right rear power window switches replacement *
  • Door glass replacement or repair
  • Power is removed from the power window master switch or any of the power window switch while the power window timer is ON.
*:
With all power windows AUTO UP/AUTO DOWN function

NOTE: If the passenger's power windows have lost power when the key off timer is ON, it cannot be operated from the driver's switch and must be reset from each passenger's power window switch.
Using the HDS (for driver's window)
  1. Connect the HDS to the data link connector.
  1. Select BODY ELECTRICAL from the SYSTEM SELECT menu.
  1. From the BODY ELECTRICAL SYSTEM SELECT menu, select POWER WINDOWS.
  1. From the MODE menu, select ADJUSTMENTS.
  1. From the ADJUSTMENT menu, select WINDOW RESET for driver's window.
  1. Follow the prompts on the screen.
  1. Confirm that the power window control unit is reset by using the power window AUTO UP and AUTO DOWN function.
Without the HDS (for driver's window)
NOTE: Do each bullet in Step 4 within 5 seconds of each other.
  1. Turn the ignition switch to ON (II).
  1. Move the power window all the way down by using the driver's power window DOWN switch.
  1. Open the driver's door.
  1. Do the following three times before going to Step 5 :
    • Turn the ignition switch to LOCK (0).
    • Push and hold the driver's power window DOWN switch.
    • Turn the ignition switch to ON (II).
    • Release the driver's power window DOWN switch.
  1. Confirm that AUTO UP no longer works. If AUTO UP still works, go back to Step 1 .
  1. Move the power window all the way down using the driver's power window DOWN switch.
  1. Pull up and hold the driver's power window UP switch until the power window is all the way up, then continue to hold the switch for 1 second.
  1. Confirm that the power window control unit is reset by using the power window AUTO UP and AUTO DOWN function. If the power window still does not work in AUTO, repeat the procedure several times, paying close attention to the 5 second time limit between steps.
Using the HDS (for passenger's window)
  1. Connect the HDS to the data link connector.
  1. Select BODY ELECTRICAL from the SYSTEM SELECT menu.
  1. From the BODY ELECTRICAL SYSTEM SELECT menu, select POWER WINDOWS.
  1. From the MODE menu, select ADJUSTMENTS.
  1. From the ADJUSTMENT menu, select WINDOW RESET for the appropriate power window.
  1. Follow the prompts on the screen.
  1. Confirm that the power window control unit is reset by using the power window AUTO UP and AUTO DOWN function.
Without the HDS (for passenger's window)
NOTE: Do each bullet in Step 4 within 5 seconds of each other.
  1. Turn the ignition switch to ON (II).
  1. Move the appropriate power window all the way down by using the DOWN switch of the power window master switch.
  1. Open the driver's door.
  1. Do the following three times before going to Step 5 :
    • Turn the ignition switch to LOCK (0).
    • Push and hold the DOWN switch of the power window master switch.
    • Turn the ignition switch to ON (II).
    • Release the DOWN switch of the power window master switch.
  1. Confirm that AUTO UP no longer works. If AUTO UP still works, go back to Step 1 .
  1. Move the appropriate power window all the way down using the appropriate power window DOWN switch.
  1. Pull up and hold the appropriate power window UP switch until the power window is all the way up, then continue to hold the switch for 1 second.
  1. Confirm that the power window control unit is reset by using the power window AUTO UP and AUTO DOWN function. If the power window still does not work in AUTO, repeat the procedure several times, paying close attention to the 5 second time limit between steps.
